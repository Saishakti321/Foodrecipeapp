Chilli Flame – Full Stack MERN Recipe App

Chilli Flame is a full-stack MERN (MongoDB, Express, React, Node.js) recipe sharing platform where users can explore recipes, create their own, rate dishes, and manage favorites securely.

The application demonstrates complete CRUD functionality, JWT-based authentication, image uploads, pagination, protected routes, and external API integration.

Key Features:-
JWT Authentication with secure password hashing (bcrypt)
Add, Edit & Delete personal recipes
Image upload using Multer
1–5 Star rating & review system
Favorite recipes (API + user-created)
Category filtering & pagination
Trending recipes based on ratings
Spoonacular API integration

Tech Stack:-
Frontend: React, React Router, Axios, CSS
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
Authentication: JWT
Security: Bcrypt
File Upload: Multer
